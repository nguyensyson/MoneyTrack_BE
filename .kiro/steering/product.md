# Product Overview

MoneyTrack_BE is a backend service for a money tracking application. This Spring Boot application provides REST APIs for financial tracking and management functionality.

## Purpose
- Personal finance tracking and management
- RESTful API backend service
- Secure financial data handling with Spring Security
- Database persistence for financial records

## Key Features
- Financial transaction tracking
- User authentication and authorization
- Data persistence with JPA/Hibernate
- RESTful API endpoints for frontend integration
